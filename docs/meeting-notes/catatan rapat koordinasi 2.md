Impromptu Zoom Meeting - April 11
VIEW RECORDING: https://fathom.video/share/EripMJzuWZdvjynB9rvN6xZuHeDnz_6N
Meeting Purpose

Review Jurnalmu.org adoption and guide new users.

Key Takeaways

  - Low Adoption: Only 3 of 21 PTMA campuses have entered journals, with Palembang leading (25), followed by Bengkulu (14) and UAD (7).
  - Dual-Approval Workflow: LPPM admins must approve both new journal manager accounts and each journal entry to maintain data quality.
  - Article Sync via OAI: Jurnalmu.org pulls articles from OJS sites using their OAI-PMH endpoint (e.g., journal.org/index.php/journal/oai). Sync failures are typically OJS-side OAI issues, not Jurnalmu.org bugs.
  - Hard Deadline: All journals from all 21 campuses must be entered and synchronized by April 30, 2026.

Topics

System Overview & Vision

  - Goal: Create a central catalog and search engine for all PTMA journals, similar to ScienceDirect.
  - Key Features:
      - LPPM Dashboard: Provides a campus-level journal map (Sinta status, fields, indexing).
      - Article Search: Pulls article metadata (title, abstract) and potentially full-text PDFs from registered journals.
      - Conference Catalog: A new module for PTMA conferences is ready for deployment this week.
      - Future Tools: Will integrate online assessment tools (like Arjuna) and accreditation support.

User Roles & Workflow

  - LPPM Admin:
      - Registers for an account, which Majelis Dikti approves.
      - Approves new journal manager accounts.
      - Approves each journal entry submitted by managers.
      - Can register journals directly.
  - Journal Manager:
      - Registers for an account, which the LPPM Admin approves.
      - Enters journal details and initiates article sync.

Live Demo & OAI Sync

  - Demo: Yasir Sidiq (UM Surakarta) registered as an LPPM Admin, demonstrating the approval workflow and journal entry process.
  - OAI-PMH Sync:
      - Purpose: Pulls article data from OJS sites.
      - Method: Add the OAI-PMH endpoint URL (e.g., journal.org/index.php/journal/oai) during journal registration.
      - Sync Trigger: Click "Sinkron Artikel" in the journal's "View" action menu.
      - Queue: Syncs are scheduled to prevent server overload.

Troubleshooting: Sync Failures

  - Problem: Palembang reported 5–6 journals failing to sync articles.
  - Cause: Sync failures are almost always due to a broken OAI-PMH endpoint on the journal's OJS site, not a bug in Jurnalmu.org.
  - Action: Report problematic journal URLs to Andri's team for diagnosis.

Next Steps

  - All 21 PTMA Campuses:
      - Register at least one LPPM Admin account this week.
      - Coordinate with faculty/prodi journal managers to register their journals.
      - Ensure all journals are entered and synchronized by April 30, 2026.
  - Andri's Team:
      - Deploy the conference catalog module this week.
      - Diagnose OAI sync issues reported by Palembang.
  - Lukman:
      - Add all participants to the Jurnalmu.org WhatsApp group for coordination.
